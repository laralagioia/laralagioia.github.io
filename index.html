Accents Available


The following accents are available on the keyboard by using these commands:


  • Acute (´)
  • Cedilla (¸)
  • Circumflex (^)
  • Diaeresis/Umlaut (¨)
  • Grave (`)
  • Inverted Question Mark (¿)
  • Inverted Exclamation Mark (¡)

Acute Accent (´)


Apostrophe key followed by the vowel

Cedilla Accent (¸)


Apostrophe key followed by 'c' key

Circumflex (^)


Circumflex key followed by the vowel

Diaeresis/Umlaut (¨)


Ctrl + Alt + colon/semi-colon key pressed at the same time followed by the vowels
or
AltGr + colon/semi-colon key pressed at the same time followed by the vowel

Grave (`)


Grave key followed by the vowel

Inverted Question Mark (¿)


Even though is not a proper portuguese accent, it might be useful to have it easily accessible.

Ctrl + Alt + question mark/foward slash key pressed at the same time
or
AltGr + foward slash/question mark key pressed at the same time

Inverted Exclamation Mark (¡)


Ctrl + Alt + 1/exclamation mark key pressed at the same time
or
AltGr + 1/exclamation mark key pressed at the same time


Installing

Download, save and extract the installation files

Alternative download links: here and here

Open folder and run the setup file


If prompted, authorize the file to make modifications

Follow the wizard and install the files on your PC


Restart your machine

Search for Region & Language settings



Click on your language Options.
You can have whatever language you wish on this step.

Click on "Add a keyboard" and select United Kingdom - PT(BR)


Your keyboard is now set and ready to use accents

Uninstalling

We're sad that the keyboard didn't work for you. Ensure that the keyboard is not selected as the default one when trying to uninstall.
You have two options to uninstall the keyboard:

Option 01: using the keyboard setup

Find the keyboard setup file and double click on it

If prompted, authorize the file to make modifications

A warning appears and the uninstalling process is compleated.



Option 02: using the "Add or remove programs" settings on Windows

Click on your windows key and search for "Add or remove programs"

Find the program on your list and select it


Click on uninstall

Allow the wizard to close applications that are using the keyboard

If no warnings appeared, you have successfully uninstalled the keyboard
